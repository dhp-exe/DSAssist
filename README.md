# DSAssist (starter)

React + Vite starter scaffold for DSAssist visualizer.

Quick start:

1. Install dependencies

```bash
npm install
# or: pnpm install
```

2. Run dev server

```bash
npm run dev
```

Notes:
- This scaffold includes a global Zustand store at `src/store/useStore.js`.
- Main layout is in `src/App.jsx` with components under `src/components`.
- A simple `SinglyLinkedListVisualizer` is provided as a template.
